# schwarzwaldcams

Schwarzwald Webcam Sammlung

## JSON Attribute

| Attribut | Beschreibung |
| --- | --- |
| `url` | URL |
| `page` | Page: Quelle, weitere Infos |
| `name` | Name, typischerweise eine Kurzbeschreibung |
| `description` | Motiv Beschreibung |
| `location` | Standort Beschreibung |
